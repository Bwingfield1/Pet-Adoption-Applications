# Pet-Adoption-Applications
Includes both Pet Adoption Websites. My very first project, which was later revamped with new languages and improved UI/UX.

## Project 1: JavaFX Scene Builder & JSON Application
**Description**
This was my very first project, where I utilized JavaFX Scene Builder to create a user interface and integrated JSON for data handling. The project served as an introduction to desktop application development and the basics of UI design.

**Features**
* Built using JavaFX for the UI
* Utilized JSON for data storage and retrieval
* Basic CRUD operations implemented
* User-friendly interface designed with JavaFX Scene Builder

**Technologies Used**
* Java
* JavaFX
* JSON

## Project 2: Angular, HTML, CSS, TypeScript, MySQL, and Node.js (Express)
**Description**
A few years later, I completely rebuilt the initial project using modern web technologies. This version features a dynamic web application built with Angular, enhanced UI/UX, and a robust backend with Node.js and MySQL.

**Features**
* Developed a responsive and modern UI with Angular, HTML, and CSS
* Implemented TypeScript for better code quality and maintainability
* Integrated Node.js with Express for the backend
* Utilized MySQL for efficient data management
* Enhanced user experience with improved design and functionality

**Technologies Used**
* Angular
* TypeScript
* HTML
* CSS
* Node.js (Express)
